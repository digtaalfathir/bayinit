BayInit

Initialize anything. Instantly.

One command turns a fresh Linux machine into a fully set-up one — no tools to install first, no backend. Each recipe is a single idempotent POSIX shell script you can read before you run.

Recipes

kiosk

Fullscreen Chrome/Chromium kiosk that autostarts on login and opens a fixed URL.

curl -fsSL https://digtaalfathir.github.io/bayinit/recipes/kiosk.sh | sh

vpn

SoftEther VPN client, imported and set to auto-connect on boot. Public script; the secret .vpn account files are pulled from a private repo with a token you supply — no token, no connection.

Profiles

A profile bundles several recipes into one deployment — one command, a fully set-up machine.

Fresh desktop → fullscreen monitoring tablet: installs Chrome, then sets up the kiosk.

read the script →

Inspect before you run

Never pipe a script into your shell without reading it first. Pipe it to a pager instead of sh: 

curl -fsSL https://digtaalfathir.github.io/bayinit/recipes/kiosk.sh | less

Pin a version

Pages URLs track the latest main. For production, pin to a tag:

curl -fsSL https://raw.githubusercontent.com/digtaalfathir/bayinit/v1.0/recipes/kiosk.sh | sh
